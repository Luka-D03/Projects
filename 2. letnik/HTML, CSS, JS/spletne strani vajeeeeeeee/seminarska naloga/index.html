Kopenske živali

Kopenske živali so živali, ki živijo pretežno ali v celoti na kopnem.

Vodne živali

Vodna žival je žival, bodisi vretenčarja ali nevretenčarja, ki večino ali celo življenje živi v vodi.

Ptiči

Ptice so skupina toplokrvnih vretenčarjev, ki sestavljajo razred Aves, za katere so značilna perja, brezzobi kljunasti čeljusti, polaganje trdo luščenih jajčec, visoka stopnja presnove, štirimestno srce in močan, a lahek okostje.

Živalski vrtovi

Živalski vrt je objekt, v katerem so živali nameščene v ograjenih prostorih, oskrbljene, razstavljene javnosti in v nekaterih primerih rejene.
